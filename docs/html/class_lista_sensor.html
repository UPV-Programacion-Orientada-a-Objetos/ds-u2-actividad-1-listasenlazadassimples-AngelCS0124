Detailed Description

template<typename T>
class ListaSensor< T >

Lista enlazada simple genérica para almacenar lecturas de sensores.

Template Parameters
TTipo de dato que almacenará la lista (int, float, etc.)

Implementa una lista enlazada simple con operaciones básicas de inserción, búsqueda, eliminación y consulta. Gestiona automáticamente la memoria mediante constructores de copia y destructores.
