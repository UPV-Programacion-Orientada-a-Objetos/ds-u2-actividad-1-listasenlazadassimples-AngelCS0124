Detailed Description

Sensor especializado para medir y procesar lecturas de presión.

Esta clase hereda de SensorBase e implementa funcionalidad específica para sensores de presión. Almacena lecturas en formato int y su procesamiento calcula el promedio de todas las lecturas.
