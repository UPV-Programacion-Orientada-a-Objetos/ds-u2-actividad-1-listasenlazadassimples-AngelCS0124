Detailed Description

Lista enlazada para gestionar una colección heterogénea de sensores.

Esta lista permite almacenar diferentes tipos de sensores (temperatura, presión, etc.) en una misma estructura mediante polimorfismo. Gestiona automáticamente la memoria de los sensores almacenados.
