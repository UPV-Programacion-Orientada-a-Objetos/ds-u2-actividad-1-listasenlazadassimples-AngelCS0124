Detailed Description

Programa principal del sistema IoT de monitoreo polimórfico.

Author
Angel Gabriel Coronado Sánchez
Date
2025

Sistema de monitoreo que lee datos de sensores desde un dispositivo ESP32 conectado por puerto serial (COM6) y los procesa mediante polimorfismo
